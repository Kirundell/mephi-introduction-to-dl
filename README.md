# Letter Classifier - EMNIST Letters

Итоговый проект по курсу «Введение в глубокое обучение» (МИФИ, 2 семестр).
Классификация рукописных букв английского алфавита A–Z на датасете **EMNIST Letters** (145 600 изображений 28×28, 26 классов).

Одна и та же MLP-архитектура реализована в **PyTorch** и **TensorFlow** для сравнения фреймворков. Дополнительно - регуляризация **Dropout(0.3)** (бонусная часть).

## Архитектура модели

| Слой | Тип    | Вход | Выход | Активация |
|------|--------|------|-------|-----------|
| 1    | Linear | 784  | 512   | ReLU      |
| 2    | Linear | 512  | 256   | ReLU      |
| 3    | Linear | 256  | 128   | ReLU      |
| 4    | Linear | 128  | 26    | Softmax   |

Параметров: ~567 K (одинаково в обоих фреймворках).

## Результаты

| Модель               | train_acc | val_acc | test_acc | gap (train − val) |
|----------------------|-----------|---------|----------|-------------------|
| PyTorch baseline     | 0.9527    | 0.9224  | **0.9154** | +0.030 (overfit)  |
| PyTorch + Dropout    | 0.8982    | 0.9155  | 0.9129   | −0.017 (норма)    |
| TensorFlow baseline  | 0.9534    | 0.9129  | 0.9093   | +0.041 (overfit)  |
| TensorFlow + Dropout | 0.8973    | 0.9145  | **0.9113** | −0.017 (норма)    |

Dropout убрал переобучение в обоих фреймворках (gap → отрицательный). На 10 эпохах test_acc практически не изменился - Dropout-модель ещё не успела доучиться.

Инференс на 5 рукописных буквах (A, C, D, F, H, нарисованных в Paint): **5/5 правильно**, средняя уверенность ~92%.

## Запуск

1. Создать окружение и установить зависимости:
   ```
   pip install torch torchvision tensorflow matplotlib numpy scikit-learn pillow
   ```
   (или использовать conda env `diplom`, если уже настроен)

2. Открыть `Шилов_КМ_Введение_в_DL.ipynb` в Jupyter / VS Code, выбрать соответствующий kernel.

3. Run All. EMNIST скачается автоматически в `./data/` при первом запуске.
   Время выполнения end-to-end: ~10–20 минут на CPU, 3–5 минут на GPU.

## Ключевые технические моменты

* **Метки EMNIST Letters имеют диапазон 1..26**, а `nn.CrossEntropyLoss` ожидает 0..25 - в train-loop вычитаем 1.
* **EMNIST хранит изображения транспонированными** (исторический артефакт). Для визуализации применяется `.T`; для пользовательских картинок в `predict_letter` тоже делается `arr.T` перед подачей в сеть - без этого предсказания становятся случайными.
* **TensorFlow датасет строится из тензоров PyTorch** - гарантирует идентичные данные в обоих фреймворках.
* `random_seed = 42` зафиксирован для Python, NumPy, PyTorch и TensorFlow.

7. Сравнение baseline vs dropout, выбор лучшей модели

10. Итоговые выводы
